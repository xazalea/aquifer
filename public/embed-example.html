Aquifer Embed Example

This page demonstrates how to embed Android games using Aquifer. Replace the APK URL in the iframe src with your own APK link.

HTML Code:

<div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden;">
  <iframe 
    src="https://aquiferx.vercel.app/embed/YOUR_APK_URL"
    style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"
    allowfullscreen
  ></iframe>
</div>
